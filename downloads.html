The L.B. Stanza compiler is currently available for x64-based systems on Linux, Macintosh, and Windows platforms. We are working hard to release Stanza for 32-bit systems, and ARM processors.

For step-by-step installation instructions please see the Getting Started chapter in Stanza By Example.
